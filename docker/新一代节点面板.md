### 新一代节点面板UIF

此面板可充当服务端和客户端，客户端使用api接口连接上服务端

- 服务端：
```
docker run --network host --name uif --privileged --restart unless-stopped -d ui4freedom/uif:latest
```


- 客户端
```
docker run --network host --name uif --privileged --restart unless-stopped -d ui4freedom/uif:latest
```
客户端使用web端口进入面板，填入服务端的`api地址和密码`，点击切换到服务端创建节点，然后切换到客户端使用这个节点
